Humble Bundle Owned Overlay

Shows which games in a bundle you already own on Steam.

17-digit number. Find yours at steamid.io. Saved between sessions. Your Steam profile and game details must be set to Public.

Get one at steamcommunity.com/dev/apikey. Not stored — keep it in a password manager and paste it here whenever you want to refresh your library.

Security: Your API key never touches the disk. It's sent once to Steam over HTTPS, the game list comes back, and the key is wiped from memory and from this input field. Only the game names are saved locally (in chrome.storage.local). No automatic refreshes — click Fetch library whenever you buy new games.
